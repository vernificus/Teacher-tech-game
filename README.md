# Teacher Tech Card Battle Game

A web app for teachers to "battle" each other in a Pokemon-style card game where cards represent instructional technology tools.

## Features

- **Card Collection**: Teachers earn cards by using tech tools with students or attending professional development
- **Battle System**: Pokemon-style turn-based battles between teachers
- **Card Management**: Track which cards each teacher owns
- **Tech Tool Database**: 16+ instructional technology tools as collectible cards

## Getting Started

1. Open `index.html` in a web browser
2. Select your teacher profile
3. View your card collection
4. Challenge another teacher to battle!

## Technology Stack

- HTML5/CSS3/JavaScript (vanilla - no framework dependencies)
- JSON data storage
- Responsive design for desktop and mobile
