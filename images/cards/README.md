## Quick Start

1. Extract each card from "EdTech Deck - All Cards NEWEST.pdf"
2. Save with the correct filename (lowercase, hyphens instead of spaces)
3. Copy all images to this directory
4. The app will automatically display your custom cards!

**Note:** If an image is missing, the app will show an emoji icon as a fallback.
